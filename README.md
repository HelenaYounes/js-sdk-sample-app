# Node Procore Boilerplate
An example app structure for a node application utilizing `node-procore`.
## Dependiences
- `brew install yarn`
- `yarn install`
- `touch .env`
```
CIENT_ID
CLIENT_SECRET
ReDIRECT_URL
SESSION_PASSWORD
```
## Development
In two separate terminal windows run:
`npm run rollup:watch`
`npm run server:watch`

## Production
1. `npm run rollup:prod`
2. `npm run server`
